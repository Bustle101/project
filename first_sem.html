Практики второго семестра

Изучение базовой структуры HTML-документа, основных тегов и их атрибутов. Рассмотрение семантической верстки и её значения для современного веб-разработчика.

Углубленное изучение возможностей HTML5, включая работу с формами, мультимедиа и canvas. Знакомство с микроформатами и метаданными.

Изучение основных принципов стилизации веб-страниц, работа с селекторами, свойствами и значениями CSS. Понимание каскадности и специфичности стилей.

Изучение современных подходов к созданию адаптивных макетов с использованием Flexbox и Grid. Практическое применение медиа-запросов для создания отзывчивого дизайна.

Знакомство с популярным CSS-фреймворком Bootstrap. Изучение компонентов, сетки и утилит для быстрой разработки современных веб-интерфейсов.

Введение в JavaScript: синтаксис, типы данных, операторы, функции. Основы программирования на JavaScript и работа с базовыми структурами данных.

Изучение Document Object Model, манипуляция элементами страницы, обработка пользовательских событий и создание интерактивных веб-приложений.

Основы работы с системой контроля версий Git, создание и управление репозиториями, публикация проектов с помощью GitHub Pages.
